# 仿写知乎项目，无框架无spring，原生java实现

## 项目说明：
  本人大一写的java项目，要求不能使用框架。阅读此项目代码最好有spring基础。
  
  项目由三个模块组成
  
    1.框架模块 framework
    
    2.业务模块 project
    
    3.前端页面
    
  本人是后端程序员，前端写得很烂
